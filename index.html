Webbutveckling Termer

Ett element är en del av ett html dokument som berättar för webbläsaren hur den ska tolka olika kommandon inom en tag/element

HTMl är en förkortning för HyperText Markup Language och är ett språk/format för att programera/bygga hemsidor

Ett unikt nummer till ett HTML elemnt för att kunna hanetera och ändra det enskilda HTML elemntet

En class (klass) är som ett ID för flera HTML element. Man kan bestämma egenskaper med klassen och sedan ge flera olika html element den klassen istället för att ge dem separata ID

"Data om data", en meta tag definerar data om ett HTML dokument och HTML dokumentets data

Responsive web design betyder att en sida är utvecklad för att kunna anpassa sig till olika format och upplösningar

Http är en förkortning för HyperText Transfer Protocol och är ett protokoll/en standrad för att dela filer och information över internet

Https är samma som Http fast kombinerat med SSL/TLS som är ett extra säkerhetslager vilket gör protokollet säkrare, s är en förkortning till secure

DNS är en förkotning för Domain Name System och konverterar domännamn till ip adresser

CDN är en förkotning för Content Delivery Network, CDN är servar utspridda i världen för att snabbare ladda hemsidor till användaren genom att ladda från den server som är närmast till användaren

Komprimerad data/kod, filen är omskriven för att göra den mindre

Local bootstrap är när man lokalt laddar ner bootstrap och länkar till den lokalt istället för att länka till bootstrap hemsidan

JQuery är ett javascript bibliotek som innehåller olika funktioner

Flexbox är en webbmodul som är designad för att placera olika objekt på en hemsida, den är baserad på två axlar och objekten kan placeras relativt till varandra och kanterna

Grid är liknande till flexbox men till skillnad från flexbox som är designat i en dimension är grid i två dimensioner. I flexbox använder man antingen row eller column men i grid kan man använda båda samtidigt

Code snippets är bitar av kod som man kan återanvända/använder ofta

WebAssembly gör det möjligt att köra olika kodspråk i en webbläsare, det är som en portal/virtuell maskin som kör koden

Chromium är en öppenkällkod som ligger till grund för många av våra moderna webbläsare. Chromium är utvecklat och underhållet av google och googles webbläsare Chrome är baserat på Chromium

Svg är ett bild filformat som använder sig av vektorer istället för pixlar, det gör det möjligt att skala upp och ner bilder/illustrationer utan att tappa någon kvalitet i upplösning.

Png24 är ett bild filformat som har stöd för upp till 24-bit färger och kan därför hålla en mycket hög kvalitet och behålla små skillnader i nyans mellan olika färger
